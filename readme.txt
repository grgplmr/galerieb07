== Description ==
Client gallery management with gated access, per-image ratings, lightbox, and admin submission emails.

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate through the Plugins screen.

== Testing ==
1. Create a new **Client Gallery**.
2. Use the metabox to copy the shortcode and share link.
3. Add images: select multiple files, then click **Start upload** (no upload happens before the button). Watch previews and error details if any file fails.
4. Insert the `[client_gallery id="123"]` shortcode in a page.
5. On the front end, enter an email to unlock the gallery. Ratings persist after refresh. Open the lightbox to rate images and navigate with keyboard arrows. Send the selection and check the admin email for the recap.
